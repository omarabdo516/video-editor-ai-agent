## قواعد التنفيذ

1. **اقرأ الملف الكامل** قبل ما تبدأ أي feature
2. **نفّذ بالترتيب** — كل feature بيعتمد على اللي قبله
3. **git commit بعد كل feature** (مش بعد كل جولة)
4. **اعمل preview** لو الملف طلب ده
5. **ما تبدأش الجولة التالية** إلا لما المستخدم يوافق
6. **لو حصل error** — صلّحه قبل ما تكمل
